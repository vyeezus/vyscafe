# vy's cafe — matcha menu app

A minimalist, high-fidelity matcha ordering experience.

## Project Structure
- index.html: Main entry point
- app.js: Logic & Cart
- style.css: Modern Styling
- svgs.js: Icon library for all menu items
- extract_svgs.js: Utility script for icon extraction
- rilakkuma.png: Peeking mascot

## How to Run
Open index.html in your browser (preferably with a Live Server).

One-time setup (live hide / show)

  1. Create a Firebase project.
  2. Enable Firestore Database (start in production mode, then paste rules below).
  3. Enable Authentication → Sign-in method → Email/Password.
  4. Add yourself: Authentication → Add user (the email must match the rule in step 6).
  5. Project settings → Your apps → Web → copy config into firebase-config.js in this repo and push.
  6. Firestore → Rules — use the template in firestore.rules.example: replace YOUR_EMAIL@example.com with your login email, then Publish.

The Firebase “API key” in the config is public by design; real protection is the Firestore rules + your password.

Sign in

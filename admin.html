Why isn’t this one click? vyscafe.com is hosted as static files on GitHub Pages — there is no server that can save changes from the browser. Your booking site almost certainly stores “open days” on another company’s servers (Calendly, Google Calendar, Squarespace form, etc.). That’s the same idea as Firebase, just under a different name.

Here you only maintain one small file, menu-config.json, in your GitHub repo. This page builds that file for you; you paste or commit it in GitHub (about the same as editing a file in your booking dashboard).
